EcoMarket

EcoMarket — це веб-додаток, який надає користувачам можливість переглядати та замовляти екологічно чисті товари, такі як екосумки. Проєкт був створений за допомогою HTML та CSS, забезпечуючи просту та інтуїтивно зрозумілу навігацію для користувачів.

Опис

У цьому проєкті представлена екосумка з льна, її опис, ціна та можливість переглядати інші товари. EcoMarket орієнтований на людей, які прагнуть до сталого способу життя та використання екологічно чистих продуктів.

Технології

- HTML5
- CSS3
- JavaScript (за потреби)

Встановлення

1. Клонуйте репозиторій:

   ```bash
   git clone https://github.com/ваш-логін/EcoMarket.git
   ```

2. Перейдіть до директорії проєкту:

   ```bash
   cd EcoMarket
   ```

3. Відкрийте файл `index.html` у вашому браузері.

## Використання

- Веб-сайт надає можливість переглядати доступні екологічні товари.
- Користувачі можуть отримати інформацію про кожен товар, включаючи опис та ціну.
